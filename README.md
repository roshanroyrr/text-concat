# Text Concatenator

A simple web tool that splits words and joins them with a custom delimiter.

## Features

- Enter text with words separated by spaces
- Choose any character as a delimiter
- See the result instantly as you type
- Responsive design for mobile and desktop

## Tech Stack

- HTML5
- CSS3 (Flexbox, CSS Variables)
- Vanilla JavaScript

## Usage

1. Open `index.html` in a browser
2. Enter your text (e.g. `apple banana cherry`)
3. Enter a delimiter (e.g. `,` or `-`)
4. See the joined result (e.g. `apple,banana,cherry`)
